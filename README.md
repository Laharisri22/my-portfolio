# 🌐 Personal Portfolio Website

This is a **personal portfolio website** designed to showcase my projects, skills, and contact information. The website highlights my work in **web development, data analysis, and machine learning projects**.

The portfolio helps demonstrate my technical skills and projects to recruiters and potential employers.

---

## 🚀 Features

* 🏠 Home section with introduction
* 💼 Projects section showcasing my work
* 🛠️ Skills section displaying technical skills
* 📧 Contact section for communication
* 📱 Responsive design for different devices
* ✨ Smooth navigation and simple UI

---

## 🛠️ Technologies Used

* **HTML** – Structure of the website
* **CSS** – Styling and layout design
* **JavaScript** – Interactive elements and animations

## 📌 Sections in the Portfolio
